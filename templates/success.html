Installation

  1. Create a directory and download the client application and license key to that directory.
  2. Rename the license key to "license.key" without the quotes.

Usage

To use the client, run either:

python client.p -p hash_file
or 
python client.py -f hash_file

When using the -p option, the hash file should be in pwdump format. When using the -f option, the hash file should be in LM:NTLM format with one hash pair per line.
